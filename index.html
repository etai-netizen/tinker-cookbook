Tinker Cookbook

Post-training with Tinker — fine-tune language models with SFT, RL, DPO, distillation & more.

⭐ Star on GitHub 📖 Documentation

🚀 Installation

  1. Sign up for Tinker here.
  2. Create an API key from the console and export it:
    export TINKER_API_KEY=your_key
  3. Install tinker-cookbook:
# Latest stable release from PyPI
uv pip install tinker-cookbook

# Or install the nightly build
uv pip install 'tinker-cookbook @ git+https://github.com/thinking-machines-lab/tinker-cookbook.git@nightly'

⚡ Tinker Primitives

The basic components to fine-tune LLMs:

import tinker
service_client = tinker.ServiceClient()
training_client = service_client.create_lora_training_client(
    base_model="meta-llama/Llama-3.2-1B", rank=32,
)
training_client.forward_backward(...)
training_client.optim_step(...)
training_client.save_state(...)
training_client.load_state(...)

sampling_client = training_client.save_weights_and_get_sampling_client()
sampling_client.sample(...)

🍳 Recipes

💬

Chat SFT

Supervised fine-tuning on conversational datasets (e.g., Tulu3).
🔢

Math RL

Reinforcement learning for mathematical reasoning with verifiable rewards.
💻

Code RL

RL on competitive programming with sandboxed code execution (DeepCoder replication).
🏆

Preference Learning

DPO and a three-stage RLHF pipeline (SFT, reward model, RL).
🎓

Distillation

On-policy and off-policy knowledge distillation with single- and multi-teacher configurations.
🔍

Tool Use

RL for retrieval-augmented generation (Search-R1 replication).
🤝

Multi-Agent

Multi-agent RL with self-play and cross-play.

📊 Evaluation (experimental)

Benchmark framework supporting 12 benchmarks: GSM8K, MATH-500, MMLU-Pro, MMLU-Redux, GPQA, IFEval, MBPP, C-Eval, SuperGPQA, IFBench, AIME 2025, AIME 2026.

from tinker_cookbook.eval.benchmarks import run_benchmarks, BenchmarkConfig

results = await run_benchmarks(
    ["gsm8k", "mmlu_pro", "ifeval"],
    sampling_client, renderer,
    BenchmarkConfig(save_dir="evals/step500"),
)
